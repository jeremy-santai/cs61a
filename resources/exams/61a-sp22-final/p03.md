LinkMaker, LinkMaker, Make Me A Link
Consider the fully implemented linkerator function:
def linkerator(num):
l = Link(num)
whole_l = l
while True:
yield whole_l
num += 1
l.rest = Link(num)
l = l.rest
A student tries using that function but encounters an error:
>>> g = linkerator(5)
>>> g[0]
Traceback (most recent call last):
File "<stdin>", line 1, in <module>
TypeError: 'generator' object is not subscriptable
(a) (1.0 pt) What line of code could they write instead of g[0] that would run without error and yield the
result Link(5)?
next(g)
(b) (1.0 pt) What is an expression that includes a call to linkerator and would cause an infinite loop? (Note
that it must be a single expression, not a statement or multiple statements).
list(linkerator(5))
