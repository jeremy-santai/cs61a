# Problem 1 — What Would Python Display?

## Page 2

1. (10.0 points)
What Would Python Display?
Assume the following code has been executed.
e = 40
def glob(e):
print(e)
e = e + 1
return e
def mid(nights):
days = 7
if days or nights:
days, nights = (days and days + 1), (nights or days)
print(days)
return nights
def special(f, x):
return f(x) + f(x)
For each expression below, write the output displayed by the interactive Python interpreter when the expression
is evaluated. The output may have multiple lines. If an error occurs, write “Error”, but include all output displayed
before the error. To display a function value, write “Function”.
(a) (2.0 pt) print(0, print(1), 2)
0 None 2
(b) (4.0 pt) print(glob(e), glob(3), e)
41 4 40
(c) (2.0 pt) mid(not 'swift')
(d) (2.0 pt) special(str, 10)
‘1010’
